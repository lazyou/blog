### fmt
* 占位符:
    * %v      相应值的默认格式
    * %+v     打印结构体时，会添加字段名. TODO: 推荐
    * %#v     相应值的Go语法表示
    * %T      相应值的类型的Go语法表示
    * %t      布尔占位符: true 或 false
    * %p      指针: 十六进制表示，前缀 0x
    * 更多参考: https://studygolang.com/articles/2644
