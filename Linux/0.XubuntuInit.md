## xubuntu 系统安装后初始化
```shell
sudo apt install git gitk curl vim tree fish nautilus htop meld vlc chromium-browser rar unrar calibre filezilla virtualbox terminator font-manager sogoupinyin


snap:
    sudo snap install phpstorm --classic
    sudo snap install goland --classic
    sudo snap install datagrip --classic

    sudo snap install vscode --classic
    sudo snap install sublime-text --classic

    sudo snap install redis-desktop-manager
    sudo snap install postman


ppa:
    sudo add-apt-repository ppa:hzwhuang/ss-qt5
    sudo apt-get update
    sudo apt-get install shadowsocks-qt5


chromium 浏览器支持插件离线安装：
chromium-browser --enable-easy-off-store-extension-install
```
