## php 调试利器 PHPDBG
* http://yzone.net/blog/142

* 不是php扩展，是单独的软件

* 安装： `sudo apt install php-phpdbg`

### 主要功能：
* 单步调试

* 灵活的下断点方式（类方法、函数、文件:行、内存地址、opcode）

* 可直接调用php的eval

* 可以查看当前执行的代码

* 用户空间API（userland/user space）

* 方便集成

* 支持指定php配置文件

* JIT全局变量

* readline支持（可选），终端操作更方便

* 远程debug，使用java GUI

* 操作简便（具体看help）
