### 使用CSS3改变文本选中的默认颜色
* https://www.cnblogs.com/yangyang63963/p/5793219.html

```css
::selection {
    background:#d3d3d3;
    color:#555;
}

::-moz-selection {
    background:#d3d3d3;
    color:#555;
}

::-webkit-selection {
    background:#d3d3d3;
    color:#555;
}
```
