## 快捷键:
```
## 编辑
Alt + 1						快速开关左侧项目目录结构

Ctrl + Shift + Enter		智能完善代码[如:if(), 好用,比如快速跳到下一行]

Ctrl + Q 					显示注释代码	(好用)
Ctrl + P 					显示显示代码传参

Ctrl + F1					显示错误或警告信息的描述 [需要把光标放到错误或警告位置]

Ctrl + O					插入覆盖父类的方法 [也算快速查看父类方法的一种]
Ctrl + I					实现抽象方法


Ctrl + /					以添加“//”的方式添加注释
Ctrl + Shift + /			添加“/**/”的方式添加注释

Ctrl + W					增量式的选中当前块	


Ctrl + Alt + L 				格式化代码 [支持选中后格式化]
Ctrl + Alt + I 				自动缩进 [必须 选中]
Ctrl + Shift + J 			合成选中代码到一行。格式化代码的反向动作


Ctrl + D 					将当前行或者选择的内容复制到下一行或光标处
Ctrl + Y 					删除光标所在的行	

Shift + Enter				另起一新行。无论光标在行的那个位置

Ctrl + Shift + U 			字符大小写切换


Ctrl + Delete 				删除光标之后的部分单词
Ctrl + Backspace 			删除光标之前的部分单词

Ctrl + +/- 					折叠/打开代码块，再次点击扩大折叠/打开范围


## 搜索/替换
F3							查找下一个，结合查找使用
Shift + F3					查找前一个，结合查找使用


## 被使用搜索
Alt + F7 / Ctrl + F7		全项目被使用查找/当前文件声明变量处
Ctrl + Alt + F7				显示详细被使用的位置列表


## 导航相关
Ctrl + Tab 					切换活动文件	[打开文件多时好用]

Ctrl + G					按行号快速定位
Ctrl + E					打开最近打开过的文件列表

Ctrl + Alt + Left/Right		返回/前进到上次导航操作
Ctrl + B or Ctrl + Click 	跳转到函数的声明处 [配合 `Ctrl + Alt + Left/Right` 很好用]

Alt + Up/Down				上下切换函数	
Ctrl + F12					打开文件结构的弹出窗	


## 重构相关
F5							复制文件
F6							移动文件

Shift + F6					为所选 变量或者文件 重命名	
```
